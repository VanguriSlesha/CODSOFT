Task 3: Recommendation System
Internship: CodSoft Python Development Internship
This is Task 3 of my CodSoft Internship — a simple Recommendation System that suggests items (movies/products/books) based on user preferences using content-based filtering.

** Features**
Simple and beginner-friendly recommendation model
Uses dictionary-based content filtering
Recommends relevant items instantly
Works on any category (Movies, Books, Fashion, etc.)

** How to Run**
Open the folder:
Task3_Recommendation_System
Run the Python file:
python recommendation_system.py
Enter a category (like: electronics, fashion, books)
Receive 5–6 recommended items

** Concepts Used**
Python functions
Dictionaries & lists
Content-based recommendation logic
Conditional statements
